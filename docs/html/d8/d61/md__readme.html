XLE

XL Token Explorer Application. Useful for browsing the output of the Parser. Either Specify File and browse AST. Or Write script inside the Program and directly execute it.

Arguments

Syntax
xle.exe <file or folder>

Examples

Example Scripts. Can be explored by dragging the whole folder on to 'xle.exe'.
